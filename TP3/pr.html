Abstract

O trabalho prático número três tem como objetivo a partir trabalho prático da semana anterior (ficheiro .dtd gerar xml) gerar paginas html.

Inicialmente o trabalho consistiu em pegar na versão do ficheiros pr.dtd padronizada feito na aula prática desta semana. Após a produção desse ficheiro, o proximo passo é o desenvolvimento do ficheiro pr.xsl. Neste ficheiro, vamos indicar como representar estilisticamente cada um dos elementos dos dados armazenados em xml, a fim de com o auxilio do oxygem xml editor representar os elementos numa página gtml.
